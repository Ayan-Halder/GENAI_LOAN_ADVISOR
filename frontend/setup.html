GenAI Loan Advisor Setup Guide

This guide will walk you through the process of setting up and running the GenAI Loan Advisor project on your local machine. Follow each step sequentially to ensure a smooth setup.

Step 1: Clone the Project Repository

Start by cloning the project repository from GitHub. Open your terminal (Command Prompt or Bash) and run the following command:

git clone https://github.com/Ayan-Halder/GENAI_LOAN_ADVISOR.git

After cloning, navigate into the project directory:

cd GENAI_LOAN_ADVISOR

Step 2: Initialize Git (if not already set up)

If this is a new project and you need to initialize Git, run the following commands:

git init
git add .
git commit -m "Initial commit"

Now, your project is initialized and ready for version control.

Step 3: Create and Activate a Virtual Environment

It's a good practice to create a virtual environment to manage dependencies. Run the following commands depending on your OS:

Windows:

python -m venv venv
venv\Scripts\activate

macOS/Linux:

python3 -m venv venv
source venv/bin/activate

Step 4: Install Project Dependencies

Once the virtual environment is activated, install the required dependencies listed in the requirements.txt file:

pip install -r requirements.txt

Step 5: Set Up Environment Variables

Create a .env file in the root directory and add your Google Generative AI API key. This key is required for the GenAI model to function properly:

# .env file content
GEMINI_API_KEY=your_google_generative_api_key

Alternative: If the .env file method does not work, set the environment variable manually:

Windows:

set GEMINI_API_KEY=your_google_generative_api_key

Linux/macOS:

export GEMINI_API_KEY=your_google_generative_api_key

Step 6: Run the Backend Server

Start the FastAPI backend server by running the following command in your terminal:

uvicorn main:app --reload

The server should now be running at http://127.0.0.1:8000/. You can access the main page by navigating to this URL in your web browser.

Step 7: Open the Web Application

Once the backend server is up and running, open your browser and go to:

http://127.0.0.1:8000/

Troubleshooting

If you encounter any issues, check the following:

  • Ensure all dependencies are installed correctly (pip list).
  • Check your .env file for the correct API key format.
  • Verify that uvicorn is running and not blocked by your firewall or any other service.
